# TASK.md — Техническое задание для Claude Code

## Задача

Построить и задеплоить ТИА (Teanium Intelligent Assistant) — AI чат-виджет для teanium.com.

---

## Что нужно создать

### 1. Vercel Edge Function (`/api/chat.js`)

Безопасный бэкенд-прокси между фронтендом и Claude API:

```
POST /api/chat
Body: { messages: [...], lang: "ru" | "en" | "ka" }
Response: { reply: "..." }
```

- Загружает `knowledge-base.md` (из GitHub raw или локально)
- Инжектирует базу знаний в system prompt
- Передаёт запрос в `claude-sonnet-4-20250514`
- API ключ хранится только в переменных окружения Vercel
- CORS настроен для `teanium.com` и `*.myshopify.com`

### 2. Виджет (`/widget/tia-widget.js`)

Один JS файл, встраиваемый через `<script>` тег в Shopify:

**Floating button:**
- Фиксированная позиция: bottom: 24px, right: 24px
- Золотая кнопка с SVG листиком чая
- Анимация появления при скролле или через 3 секунды

**Chat panel (открывается по клику):**
- Дизайн строго по `DESIGN.md` — воспроизвести 1-в-1
- Приветственное сообщение на языке браузера (auto-detect)
- Quick prompts на первом экране (4 кнопки)
- Streaming ответов (печатается по мере генерации)
- Сохранение истории в sessionStorage (в рамках визита)
- Кнопка закрытия (X) в хедере

### 3. Конфигурация (`vercel.json`)

```json
{
  "functions": {
    "api/chat.js": {
      "runtime": "edge"
    }
  },
  "headers": [...]
}
```

### 4. Инструкция по деплою (`DEPLOY.md`)

Пошаговая инструкция:
1. `git init` и push на GitHub
2. Подключить репозиторий к Vercel
3. Добавить `ANTHROPIC_API_KEY` в Vercel Environment Variables
4. Деплой одной командой
5. Встроить `<script>` тег в Shopify theme.liquid

## Архитектурные вопросы для Claude Code

> **Claude Code, прошу дать своё мнение по следующим вопросам:**

### Вопрос 1: Vercel vs альтернативы
Предварительный выбор — Vercel Edge Functions. Подтвердишь ли это для данного случая (Shopify + одна функция + бесплатный tier)? Или есть лучший вариант (Cloudflare Workers, Netlify Functions)?

### Вопрос 2: Floating button vs встроенный блок
Предварительный выбор — плавающая кнопка в углу. Учитывая что это Shopify магазин с мобильным трафиком — это оптимально? Или стоит рассмотреть встроенный блок на конкретных страницах?

### Вопрос 3: Streaming
Стоит ли реализовать streaming ответов (Server-Sent Events) для лучшего UX, или обычный fetch достаточен для данного виджета?

### Вопрос 4: База знаний
Лучше загружать knowledge-base.md при каждом запросе (свежие данные, чуть медленнее) или кешировать на стороне Vercel (быстро, обновляется раз в N минут)?
